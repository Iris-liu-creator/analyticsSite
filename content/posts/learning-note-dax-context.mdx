## The mental shift

DAX becomes easier when you stop thinking only in cells and start thinking in filters. A measure answers a question inside the current filter context.

## Row context

Row context is about the current row while evaluating calculated columns or iterators. It is useful, but it does not automatically filter the model the way many beginners expect.

## Filter context

Filter context comes from slicers, relationships, visuals, and explicit filter expressions. Most dashboard behavior comes from understanding this layer.

## My practical reminder

When a measure surprises me, I ask: what filters are active, where did they come from, and did I intentionally change them?
